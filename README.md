# 🎵 Spotify Clone

A simple frontend **Spotify clone** built using **HTML** and **CSS**.  
This project replicates the look of Spotify’s web interface.

## 🚀 Features

- Clean UI inspired by Spotify
- Responsive layout using CSS


## 🛠️ Tech Stack

- HTML5
- CSS3
